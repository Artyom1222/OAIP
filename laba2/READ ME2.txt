Реализовать очередь Unemployed на бирже труда. Один элемент
списка безработных имеет поля: фамилия, возраст, пол, образование, профессия,
должность, стаж работы в последней должности и общий стаж, дата постановки
на учет, желаемая заработная плата, желаемая должность. Для обоих списков
реализовать обычные операции добавления и удаления. Реализовать операцию
генерации конкретного объекта Group (группа), содержащего список
очередников, из объекта типа Unemployed. Должна быть обеспечена
возможность выбирать группу по любому из полей класса Unemployed.

Алгоритм
1.Создание steckUnemployedStart, steckUnemployedEnd, group
2.Вход в бесконечный цикл
3.Вывод информации в консоль
4.Ожидание нажатия клавиши
5.Если нажата клавиша 1, то вызываем функцию add с параметром 0, для добавления в начало очереди
6.Если нажата клавиша 2, то вызываем функцию add с параметром 1, для добавления в конец очереди
7.Если нажата клавиша 3, то вызываем функцию del с параметром 0, для удаления в начале очереди
8.Если нажата клавиша 4, то вызываем функцию del с параметром 1, для удаления в конце очереди
9.Если нажата клавиша 5, то вызываем функцию seeFromStart, которая выведит очередь с начала до конца
10.Если нажата клавиша 6, то вызываем функцию seeFromEnd, которая выведит очередь с конца до начала
11.Если нажата клавиша 7, то вызываем функцию addInGroup, где пользователь вводит название группы и по какому параметру добовлять
Unemployed в группу 
12.Если нажата клавиша 8, то вызываем функцию seeGroup
13.Если нажата клавиша 0, то завершаем программу
